üåê︎GeoJSON to GML converter
GeoJSON works: it succinctly describes geometric features in a way both humans and machines can understand. As a consequence, GeoJSON has become one of the leading data formats in GIS, and since it is a species of JSON, it naturally rules online GIS visualization. However, the OGC has mainatined another standard, GML since the '90s [citation]. I wrote a javascript module to translate GeoJSON to GML to allow web GIS technologies to better talk to older tools that rely on OGC GML, i.e. Geoserver.
OpenLayers and Leaflet-WFST can translate GeoJSON to GML. Both are full-fledged web GIS tools. This is only a string formatting library to translate GeoJSON to GML so that it should plug-and-play in any context.

üîñ︎Scraping Mozilla Developer Network compatibility tables
I wrote a bookmarklet to scrape MDN's old browser compatibility tables. To use it:
  1. Save it by (chrome) dragging it to your bookmarks bar or (firefox) right-clicking > clicking "bookmark this link".
  2. On a MDN documentation page which needs modernization, click the bookmarklet in your bookmarks menu/bar.
  3. Check the scraped data in the JS console, including where you should submit it as a PR.
